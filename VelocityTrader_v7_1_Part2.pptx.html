Signal Generation & Threshold Logic

BREAKOUT Regime

Trades acceleration spikes exceeding agent threshold
if acceleration > threshold:
  direction = sign(acceleration)
Sniper: 1.5σ threshold (selective)
Berserker: 1.0σ threshold (aggressive)

TREND Regime

Trades sustained velocity with regime-specific multipliers
if |accel| > threshold × trend_mult
  and velocity_persist >= 5:
  direction = sign(velocity)

Default: 0.8x (more permissive)

Fades price extremes with confirmation from deceleration

DAMPING RATIO ANALYSIS

SymC Monitor: Damping Ratio Analysis

Chi (χ) Calculation

Damping Ratio Formula
χ = Liquidity / Flow
Where:
• Liquidity = average order book depth
• Flow = average trading volume
Exponential Moving Average
Both components smoothed with 0.95 decay:
Liquidity_t = 0.95 × L_t-1 + 0.05 × L_current
Flow_t = 0.95 × F_t-1 + 0.05 × F_current

Chi Interpretation

High χ > 1.3: Thick, stable market
Medium χ 0.7-1.3: Balanced
Low χ < 0.7: Thin, reactive market

Q-Learning Implementation

Q-Value Structure
Each agent maintains 9 Q-values:
BREAKOUT
qBuy, qSell, qHold
TREND
qBuy, qSell, qHold
MEANREV
qBuy, qSell, qHold
Q-Update Formula
Q(s,a) += α × [r - Q(s,a)]
Where:
• α = adaptive learning rate
• r = shaped reward (asymmetrical)
Action Selection: Choose action with highest Q-value for regime

Learning Rate Schedule

surprise = |reward - Q(s,a)|
boost = min(1 + surprise × 0.1, max_boost)
alpha_effective = alpha × boost
When outcomes exceed expectations, learning accelerates up to 2x

Reward Formula

if net_pnl >= 0:
  reward = net_pnl
else:
  reward = net_pnl * loss_penalty

reward -= time_decay * duration
reward = max(reward, -max_loss)

Time Decay & Friction

Time Decay
Penalizes duration to encourage timely decisions:
decay = duration_min × 0.01
RL-tunable: 0 to 0.05 per minute
Friction Costs
Always subtracted for realism:
Spread Cost = (spread/tick_size) × tick_value × lots
Commission = commission × lots × 2
Total Friction = spread + commission
Result: Realistic transaction cost modeling

Surprise Boost

if |reward - Q(s,a)| > threshold:
  boost = 1 + surprise × 0.1
  reward *= min(boost, 2.0)
